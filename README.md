# IoT Platform Project

This project is a comprehensive IoT platform that aims to connect various IoT devices and manage them efficiently. The platform is built using Python and FastAPI, and it integrates with Modbus protocol for device communication. 

## Features
- Device management
- Modbus TCP communication
- RESTful API service
- Task scheduling
- Database management
- Utility functions

## Getting Started

To get started with the project, ensure you have Python installed and then install the required dependencies.
